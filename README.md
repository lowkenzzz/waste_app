# Waste App

A web application designed to help users manage and track waste disposal efficiently. Waste App aims to promote sustainable practices by making waste management easy and engaging.

---

## 🚀 Features

- **Record Waste Disposal:** Log everyday waste or recycling events.
- **Manage Categories:** Track different types of waste (e.g., plastic, paper, organic).
- **Statistics & Reports:** Visualize your waste trends with charts.
- **Responsive Design:** Mobile-friendly interface.
- **User Account Support:** (If implemented) Sign up/log in to save progress.

---

## 🛠️ Tech Stack

- **JavaScript:** Main scripting language (frontend logic)
- **CSS:** Custom styles and responsive layout
- **HTML:** Markup structure

## 📦 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/lowkenzzz/waste_app.git
cd waste_app
```

### 2. Install Dependencies

If your app uses Node.js/npm (for build tools, package management, or frameworks like React/Vue):

```bash
npm install
```

<details>
<summary>If you do not use npm or a build step, you can skip this.</summary>

**Just open `index.html` directly in your web browser.**
</details>

### 3. Start the Application

#### If using a local development server (recommended for JS modules, APIs, or SPAs):

```bash
npm start
```
or
```bash
npm run dev
```
(Choose the script configured in your `package.json`, e.g., with tools like [lite-server](https://www.npmjs.com/package/lite-server), [live-server](https://www.npmjs.com/package/live-server), or [Vite](https://vitejs.dev/))

#### If just using static files:

- Open `index.html` in your browser.
- Or run a static server:
  ```bash
  npx serve .
  ```
  or
  ```bash
  python3 -m http.server
  ```
  and visit [http://localhost:8000](http://localhost:8000)
